Simple web application made with **Flask**, using a **virtual environment** to isolate dependencies.
This project is part of the "Be My DevOps" exercise.

---

## Requirements
- Python 3.9 or newer
- pip
- python3-venv

---

## Setup

Clone the repository:
```bash
git clone https://github.com/TSoile/be-my-devops
cd be-my-devops
```
Create virtual environment:
```bash
python3 -m venv venv
```
Activate virtual environment:
```bash
source venv/bin/activate
```
Install dependencies:
```bash
pip install -r requirements.txt
```
Run:  
```bash
python app.py
```
Usage

Open your browser at:

http://localhost:5000
